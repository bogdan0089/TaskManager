# Task Manager API

**Task Manager API** is a FastAPI project for managing users and tasks with async PostgreSQL, JWT authentication, and basic roles (`admin` / `worker`).

---

## Technologies

- Python 3.11  
- FastAPI  
- SQLAlchemy (async)  
- PostgreSQL  
- Docker / Docker Compose  
- JWT authentication  
- Pydantic schemas  

---
