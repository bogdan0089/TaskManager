# Task Manager API

**Task Manager API** is a FastAPI project for managing users and tasks with async PostgreSQL, JWT authentication, and basic roles (`admin` / `worker`).

---

## Technologies

- Python 3.11  
- FastAPI  
- SQLAlchemy ORM (async)  
- PostgreSQL  
- Docker / Docker Compose  
- JWT 
- Pydantic schemas
- Alembic
- Clean code
- Unit of Work pattern
- Pytest
